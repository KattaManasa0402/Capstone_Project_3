Problem Description

Traditional cybersecurity training methods, such as static presentations and generic quizzes, often fail to engage employees effectively, leading to poor knowledge retention and increased vulnerability to cyber threats. Our project addresses this issue by developing an engaging, Generative AI-powered game that delivers dynamic, realistic, and interactive training experiences.

Objectives

  • Gamified Platform with Unity: Develop a visually appealing, interactive game using Unity's free Personal Edition, featuring an intuitive interface and engaging gameplay to maximize user engagement.
  • Dynamic Q&A with Generative AI: Integrate an open-source Generative AI model (e.g., Llama 3 or Mistral) to create unique, challenging scenarios like phishing emails or social engineering scams for non-repetitive training.
  • Rewarding System: Implement a points, badges, and level-unlock system to motivate users and encourage continuous learning through rewarding correct responses.
  • Real-time Feedback: Provide instant feedback after each interaction, explaining correct and incorrect choices to facilitate learning in a safe environment.
  • Performance Tracking: Track user progress and performance over time to offer personalized insights into strengths and weaknesses for a tailored learning experience.

Technology Stack

The project leverages a robust stack of free, open-source tools to ensure scalability and accessibility.

  • Game Engine: Unity (Personal Edition) with C# for cross-platform game development and logic scripting.
  • Backend: Python with Flask or FastAPI for server logic, API handling, and AI communication.
  • Generative AI: Open-source models like Llama 3 or Mistral, deployable via free APIs or local servers.
  • Database: PostgreSQL for storing user profiles, progress, scores, and game data.

Innovation

Our project innovates by integrating a dynamic Generative AI engine into an interactive game environment, offering:

  • Infinite Scenario Variability: AI-generated unique scenarios prevent rote memorization, teaching users to recognize and respond to cyber threat principles.
  • Personalized Threat Modeling: The system analyzes user weaknesses and creates targeted challenges for a tailored, effective learning journey.
