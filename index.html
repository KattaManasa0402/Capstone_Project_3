Problem Description

Traditional training methods for cybersecurity, such as static presentations and generic quizzes, have proven ineffective in keeping employees engaged and prepared for the ever-evolving landscape of cyber threats. This results in poor knowledge retention and a high risk of human-error-induced security breaches. Our project addresses this critical gap by creating a highly engaging and educational game that uses Generative AI to deliver a dynamic and realistic training experience.

Objectives

  • Gamified Platform with Unity: We will develop a visually appealing and interactive game using the free Personal Edition of Unity. The game will be the core platform for our training, featuring an intuitive interface and compelling gameplay to maximize user engagement.
  • Dynamic Q&A with Generative AI: The game will be powered by a free and open-source Generative AI model (like Llama 3 or Mistral). This AI will dynamically create unique and challenging scenarios, such as sophisticated phishing emails or social engineering scams, ensuring the training is never repetitive.
  • Rewarding and Motivating System: Users will be rewarded with points, badges, and the ability to unlock new levels or content for correctly identifying and responding to threats. This reward system is designed to keep players motivated and encourage continuous learning.
  • Real-time Learning and Feedback: After each interaction, the game will provide instant feedback, explaining the correct and incorrect choices. This educational approach helps users learn from their mistakes immediately in a safe environment.
  • Performance Tracking: We will implement a system to track each user's progress and performance over time. This data will provide valuable insights into individual strengths and weaknesses, allowing for a truly personalized learning journey.

Technology Stack

Our project will be built using a stack of powerful, free, and open-source tools to ensure a robust and scalable solution.

  • Game Engine: Unity (Personal Edition) for cross-platform game development and C# for scripting the game logic.
  • Backend: Python with the Flask or FastAPI framework to handle game server logic, API requests, and communication with the AI model.
  • Generative AI: An open-source Large Language Model like Llama 3 or Mistral, which can be deployed via a free API or on a local server.
  • Database: PostgreSQL, a powerful and free relational database, for storing user profiles, progress, scores, and game data.

Innovation

Our primary innovation is the seamless integration of a dynamic Generative AI engine within a fun, interactive game environment. This moves beyond traditional gamification to create an intelligent and adaptive training system that provides:

  • Infinite Scenario Variability: The AI's ability to generate countless unique scenarios prevents rote memorization, teaching users to recognize and respond to the underlying principles of cyber threats.
  • Personalized Threat Modeling: The system can analyze user weaknesses and dynamically create challenges that specifically target those areas, ensuring a tailored and effective learning journey for every player.
